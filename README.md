# Lotto Signal Lab

한국 로또 6/45의 역대 당첨 데이터를 분석하고 50,000회 가중 모의 추첨을 수행하는 웹 앱입니다.

## 주요 기능

- 최신 회차·당첨번호·보너스·1등 정보 표시
- 1회부터 최신 회차까지의 전체 이력 번들 캐시
- 온라인 최신 회차 자동 확인 및 브라우저 `localStorage` 백업
- 온라인 연결 실패 시 마지막 저장 데이터 사용
- 장기 빈도, 최근 52회 빈도, 미출현 간격, 번호 쌍 결합도 분석
- 50,000회 가중 모의 추첨
- 종합 점수 상위 5세트와 개별 번호 TOP 12 표시
- 번호 출현 분포 및 조합 구조 시각화
- PC·태블릿·모바일 반응형 UI

## 로컬 실행

```bash
cd C:/Users/yelim/lotto-signal-lab
npm start
```

브라우저에서 `http://127.0.0.1:4173`을 엽니다.

## 데이터 갱신

```bash
npm run update-data
```

`data/lotto-history.json`이 최신 회차까지 갱신됩니다.

## 테스트

```bash
npm test
```

## 데이터와 확률 고지

온라인 최신 데이터는 공개 데이터 미러 `https://smok95.github.io/lotto/results`를 사용하며 마지막 성공 데이터를 프로젝트와 브라우저에 저장합니다. 로또 6/45의 모든 조합은 실제 추첨에서 동일한 `1 / 8,145,060` 확률을 가집니다. 이 앱의 선택률은 모델 내부의 50,000회 시뮬레이션 빈도이며 당첨을 예측하거나 보장하지 않습니다.
